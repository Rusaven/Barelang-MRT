# Barelang-MRT
Website for team profile of Barelang MRT.

Hi everyone. My name is Aditya Gading Saputra. I'm the person who responsible for making this website.

Barelang MRT is a robotics team from Politeknik Batam. Barelang MRT is part of BRAIL (Barelang Artificial Intelligence Lab). A place where every robotics team with different category doing their project. Barelang MRT is a robotics team that focussing on surface and underwater robot that can move freely by itself. We already followed several competition in the past. Our greatest achievment is 2nd place on national for underwater robot competition.
